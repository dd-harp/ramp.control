# Set the bednet

Set the value of exogenous variables related to bednet

## Usage

``` r
# S3 method for class 'static'
Bed_Net_2(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  state variables vector

- xds_obj:

  a **`ramp.xds`** model object

## Value

a **`ramp.xds`** model object
