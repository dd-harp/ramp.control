# Bed Net Effect Sizes

Compute bed net effect sizes

## Usage

``` r
Bed_Net_Effect_Sizes(t, y, xds_obj, s = 1)
```

## Arguments

- t:

  current simulation time

- y:

  state variables vector

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

## Value

the **`xds`** model object
