# IRS Contact

Set up a linear model for contact relative to coverage

## Usage

``` r
# S3 method for class 'linear'
IRS_Contact(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  state variables

- xds_obj:

  a **`ramp.xds`** model object

## Value

a **`ramp.xds`** model object
