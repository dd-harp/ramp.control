# IRS Contact for Multiround Models

A model for IRS contact over time when there have been several rounds of
IRS

## Usage

``` r
# S3 method for class 'multiround'
IRS_Contact(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  state variables

- xds_obj:

  a **`ramp.xds`** model object

## Value

a **`ramp.xds`** model object
