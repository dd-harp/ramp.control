# IRS Coverage with Multiple Rounds

A model for IRS coverage over time when there have been several,
different IRS models

## Usage

``` r
# S3 method for class 'multiround'
IRS_Coverage(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  state variables

- xds_obj:

  a **`ramp.xds`** model object

## Value

a **`ramp.xds`** model object
