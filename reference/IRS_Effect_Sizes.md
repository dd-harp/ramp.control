# IRS effect_sizes

Set the value of variables to model the effect_sizes of IRS

## Usage

``` r
IRS_Effect_Sizes(t, y, xds_obj, s = 1)
```

## Arguments

- t:

  current simulation time

- y:

  state variables

- xds_obj:

  a **`ramp.xds`** model object

- s:

  the vector species index

## Value

a **`ramp.xds`** model object
