# Set no use_bednets

The null model for use_bednets

## Usage

``` r
# S3 method for class 'none'
IRS_Effect_Sizes(t, y, xds_obj, s = 1)
```

## Arguments

- t:

  current simulation time

- y:

  state variables

- xds_obj:

  a **`ramp.xds`** model object

- s:

  the vector species index

## Value

[list](https://rdrr.io/r/base/list.html)
