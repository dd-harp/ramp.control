# Dynamic Vector Control

Calls `dynamic_vector_control`

## Usage

``` r
# S3 method for class 'dynamic'
VectorControl1(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

## Value

a named [list](https://rdrr.io/r/base/list.html)

## Note

This the function that implements various modes of vector control.
