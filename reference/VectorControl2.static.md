# Distribute vector control, the null model

Implements
[ramp.xds::VectorControl1](https://dd-harp.github.io/ramp.xds/reference/VectorControl1.html)
for the control model of vector control (do nothing)

## Usage

``` r
# S3 method for class 'static'
VectorControl2(t, y, xds_obj)
```

## Arguments

- t:

  current simulation time

- y:

  state vector

- xds_obj:

  an **`xds`** model object

## Value

a named [list](https://rdrr.io/r/base/list.html)
