# Set Up Mass Treatment Events

Set Up Mass Treatment Events

## Usage

``` r
add_mass_treat_events(xds_obj, jdates, span, frac_tot, test)
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- jdates:

  the julian dates of mass treatment events

- span:

  the treatment period

- frac_tot:

  the fraction treated

- test:

  FALSE for mass drug administration; TRUE for mass screen and treat

## Value

a **`ramp.xds`** model object
