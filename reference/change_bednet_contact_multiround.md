# Setup Muti-Round bednet Contact

With multi-round bednet contact, there is a different relationship
between contact and contact in each round

## Usage

``` r
change_bednet_contact_multiround(xds_obj, contact)
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- contact:

  the new contact parameter

## Value

a **`ramp.xds`** model object
