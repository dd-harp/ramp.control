# Change bednet Shocks

For `eir` models, set up a multi-round sharkbite function to model
effect sizes of bednet on the EIR as a perturbation.

## Usage

``` r
change_bednet_shock_multiround(xds_obj, shock)
```

## Arguments

- xds_obj:

  an **`xds`** model object

- shock:

  the scaling parameter

## Value

an **`xds`** model object
