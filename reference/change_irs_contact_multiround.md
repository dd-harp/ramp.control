# Setup Muti-Round IRS Contact

With multi-round irs contact, there is a different relationship between
coverage and contact in each round

## Usage

``` r
change_irs_contact_multiround(xds_obj, contact)
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- contact:

  the new contact parameter

## Value

a **`ramp.xds`** model object
