# Set Up the Bed Net Access Object

Set up an object to model bed net ownership

## Usage

``` r
setup_bednet_access(name, xds_obj, options = list())
```

## Arguments

- name:

  the model name

- xds_obj:

  a **`ramp.xds`** model object

- options:

  a list of options to override defaults

## Value

a **`ramp.xds`** model object
