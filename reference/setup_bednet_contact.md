# Set Up the Bed Net Contact Object

Set up an object to model bed net effective contact.

## Usage

``` r
setup_bednet_contact(name, xds_obj, options = list())
```

## Arguments

- name:

  the model name

- xds_obj:

  a **`ramp.xds`** model object

- options:

  a list of options to override defaults

## Value

a **`ramp.xds`** model object
