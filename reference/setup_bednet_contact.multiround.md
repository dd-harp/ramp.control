# Setup Muti-Round Bed Net Contact

With multi-round bed net contact, there is a different relationship
between contact and contact in each round

## Usage

``` r
# S3 method for class 'multiround'
setup_bednet_contact(name = "multiround", xds_obj, options = list())
```

## Arguments

- name:

  the model name

- xds_obj:

  a **`ramp.xds`** model object

- options:

  a list of options to override defaults
