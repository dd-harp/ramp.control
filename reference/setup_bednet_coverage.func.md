# Set Up a Bed Net Coverage Function

This sets up a function to model bed net coverage.

- A function F_cover

## Usage

``` r
# S3 method for class 'func'
setup_bednet_coverage(name = "func", xds_obj, options = list())
```

## Arguments

- name:

  the model name

- xds_obj:

  a **`ramp.xds`** model object

- options:

  a list of options to override defaults
