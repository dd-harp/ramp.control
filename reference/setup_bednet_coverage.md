# Set Up the Bed Net Coverage Object

Set up an object to model bed net effective coverage, which is defined
as the product of access and use.

## Usage

``` r
setup_bednet_coverage(name, xds_obj, options = list())
```

## Arguments

- name:

  the model name

- xds_obj:

  a **`ramp.xds`** model object

- options:

  a list of options to override defaults

## Value

a **`ramp.xds`** model object
