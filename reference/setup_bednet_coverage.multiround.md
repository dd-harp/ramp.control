# Set Up a Bed Net Coverage Function

This sets up a function to set the value of bed net coverage

## Usage

``` r
# S3 method for class 'multiround'
setup_bednet_coverage(name = "multiround", xds_obj, options = list())
```

## Arguments

- name:

  the model name

- xds_obj:

  a **`ramp.xds`** model object

- options:

  a list of options to override defaults
