# Set Up the Null Bed Net Coverage Object

Set up the null bed net coverage object

## Usage

``` r
# S3 method for class 'none'
setup_bednet_coverage(name, xds_obj, options = list())
```

## Arguments

- name:

  the name of a model to set up

- xds_obj:

  a **`ramp.xds`** model object

- options:

  a list of options to override defaults

## Value

a **`ramp.xds`** model object
