# Set up dynamic forcing

If dynamic forcing has not already been set up, then turn on dynamic
forcing and set all the

## Usage

``` r
# S3 method for class 'lemenach'
setup_bednet_effect_sizes(name, xds_obj, s = 1, options = list())
```

## Arguments

- name:

  the bed net effect sizes model name

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- options:

  a list of options to override defaults
