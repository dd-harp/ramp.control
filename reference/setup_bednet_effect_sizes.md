# Set Up the Bed Net Effect Sizes Object

Set up a model object to compute an effect sizes of bed nets

## Usage

``` r
setup_bednet_effect_sizes(name, xds_obj, s = 1, options = list())
```

## Arguments

- name:

  the bed net effect sizes model name

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- options:

  a list of options to override defaults

## Value

the **`xds`** model object
