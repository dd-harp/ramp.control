# Set Up the Null Model for Bed Net Effect Sizes

Set up the null bed net effect sizes model object

## Usage

``` r
# S3 method for class 'none'
setup_bednet_effect_sizes(name, xds_obj, s = 1, options = list())
```

## Arguments

- name:

  the bed net effect sizes model name

- xds_obj:

  an **`xds`** model object

- s:

  the vector species index

- options:

  a list of options to override defaults

## Value

the **`xds`** model object
