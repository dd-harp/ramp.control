# Set Up the Bed Net Effects Object

Set up an object to compute an effect of bed nets, such as to modify the
availability of human hosts for blood feeding

## Usage

``` r
setup_bednet_effects(name, xds_obj, options = list())
```

## Arguments

- name:

  the model name

- xds_obj:

  an **`xds`** model object

- options:

  a list of options to override defaults

## Value

the **`xds`** model object
