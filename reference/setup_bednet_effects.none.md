# Set Up the Null Bed Net effectsership Object

Set up the null bed net effectsership object

## Usage

``` r
# S3 method for class 'none'
setup_bednet_effects(name, xds_obj, options = list())
```

## Arguments

- name:

  the name of a model to set up

- xds_obj:

  an **`xds`** model object

- options:

  a list of options to override defaults

## Value

the **`xds`** model object
