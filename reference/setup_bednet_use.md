# Set Up the Bed Net Use Object

Set up an object to model bed net use

## Usage

``` r
setup_bednet_use(name, xds_obj, options = list())
```

## Arguments

- name:

  the model name

- xds_obj:

  an **`xds`** model object

- options:

  a list of options to override defaults

## Value

the **`xds`** model object
