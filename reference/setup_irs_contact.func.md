# Set Up a IRS Contact Function

This sets up a function to set the value of bed net contact

## Usage

``` r
# S3 method for class 'func'
setup_irs_contact(name = "func", xds_obj, options = list())
```

## Arguments

- name:

  the model name

- xds_obj:

  a **`ramp.xds`** model object

- options:

  a list of options to override defaults
