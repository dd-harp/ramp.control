# Set Up the Null IRS Contact Object

Set up the null irs coverage object

## Usage

``` r
# S3 method for class 'linear'
setup_irs_contact(name, xds_obj, options = list(cp = 1))
```

## Arguments

- name:

  the name of a model to set up

- xds_obj:

  a **`ramp.xds`** model object

- options:

  a list of options to override defaults

## Value

a **`ramp.xds`** model object
