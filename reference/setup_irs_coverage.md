# Set Up the IRS Coverage Object

Set up an object to model the effects of IRS, usually through the
computation of some intermediate variable

## Usage

``` r
setup_irs_coverage(name, xds_obj, options = list())
```

## Arguments

- name:

  the model name

- xds_obj:

  a **`ramp.xds`** model object

- options:

  a list of options to override defaults

## Value

a **`ramp.xds`** model object
