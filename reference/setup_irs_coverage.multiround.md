# Set up dynamic forcing

A set up utility function for
[IRS_Coverage](https://dd-harp.github.io/ramp.control/reference/IRS_Coverage.md).

## Usage

``` r
# S3 method for class 'multiround'
setup_irs_coverage(name, xds_obj, options = list())
```

## Arguments

- name:

  the model name

- xds_obj:

  a **`ramp.xds`** model object

- options:

  a list of options to override defaults

## Value

a IRS coverage model object
