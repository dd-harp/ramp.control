# Set Up the IRS effect_sizes Object

Set up an object to model the effect_sizes of IRS, usually through the
computation of some intermediate variable

## Usage

``` r
setup_irs_effect_sizes(name, xds_obj, s = 1, options = list())
```

## Arguments

- name:

  the model name

- xds_obj:

  a **`ramp.xds`** model object

- s:

  the vector species index

- options:

  a list of options to override defaults

## Value

a **`ramp.xds`** model object
