# Set Up the Null IRS effect_sizes Object

Set up a model for IRS effect_sizes that does nothing.

## Usage

``` r
# S3 method for class 'none'
setup_irs_effect_sizes(name, xds_obj, s = 1, options = list())
```

## Arguments

- name:

  the model name

- xds_obj:

  a **`ramp.xds`** model object

- s:

  the vector species index

- options:

  a list of options to override defaults

## Value

a **`ramp.xds`** model object
