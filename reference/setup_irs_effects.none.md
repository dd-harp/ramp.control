# Set Up the Null IRS Effects Object

Set up a model for IRS effects that does nothing.

## Usage

``` r
# S3 method for class 'none'
setup_irs_effects(name, xds_obj, options = list())
```

## Arguments

- name:

  the name of a model to set up

- xds_obj:

  a **`ramp.xds`** model object

- options:

  a list of options to override defaults

## Value

a **`ramp.xds`** model object
