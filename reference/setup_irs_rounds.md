# Set up dynamic forcing

Set up the IRS rounds

## Usage

``` r
setup_irs_rounds(xds_obj, mx, as_shock = FALSE)
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- mx:

  the maximum

- as_shock:

  set the function class to `sharkfin` (if `FALSE`) or `sharkbite` (if
  `TRUE`)

## Value

set up the rounds
