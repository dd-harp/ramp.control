# Set Up the House Spraying Object

Set up an object to model house spraying

## Usage

``` r
setup_spray_houses(name, xds_obj, options = list())
```

## Arguments

- name:

  the model name

- xds_obj:

  a **`ramp.xds`** model object

- options:

  a list of options to override defaults

## Value

a **`ramp.xds`** model object
