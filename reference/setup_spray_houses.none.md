# Set Up the Null House Spraying Object

Set up the null house spraying model object

## Usage

``` r
# S3 method for class 'none'
setup_spray_houses(name, xds_obj, options = list())
```

## Arguments

- name:

  the name of a model to set up

- xds_obj:

  a **`ramp.xds`** model object

- options:

  a list of options to override defaults

## Value

a **`ramp.xds`** model object
