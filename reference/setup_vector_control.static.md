# Set Up Vector Control

If `class(xds_obj$vector_control_obj) == 'none'` this function sets up
the objects for all implemented modes of vector control.

## Usage

``` r
# S3 method for class 'static'
setup_vector_control(xds_obj)
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

## Value

a **`ramp.xds`** model object
