# Show bednet Events

Show bednet Events

## Usage

``` r
show_bednet_events(
  xds_obj,
  mn = 0,
  mx = 1,
  laby = 0.1,
  clr = "#E4460AFF",
  add = FALSE
)
```

## Arguments

- xds_obj:

  a **`ramp.xds`** model object

- mn:

  the bottom of the segment

- mx:

  the top of the segment

- laby:

  the height of the labels

- clr:

  a color for the line segments

- add:

  if TRUE, add to an existing plot

## Value

a **`ramp.xds`** model object
