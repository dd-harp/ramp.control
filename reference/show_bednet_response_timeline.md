# Make Parameters for an bednet Round

Return the parameters to call a sharkfin function to model a single
round of bednet.

## Usage

``` r
show_bednet_response_timeline(d_50, d_shape, clr = "black", add = FALSE)
```

## Arguments

- d_50:

  the half-saturation parameter

- d_shape:

  the shape parameter

- clr:

  the line color

- add:

  if TRUE, add to an existing plot

## Value

a **`xds`** object
