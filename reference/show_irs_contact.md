# Plot IRS Contact

Plot IRS Contact

## Usage

``` r
show_irs_contact(tt, xds_obj, clr = "black", add = FALSE)
```

## Arguments

- tt:

  a set of time points

- xds_obj:

  a **`ramp.xds`** model object

- clr:

  plotting color

- add:

  add to an existing plot

## Value

a **`ramp.xds`** xds_obj object
