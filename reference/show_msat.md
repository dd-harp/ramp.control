# Show MSAT

Plot the per-capita mass treatment rate

## Usage

``` r
show_msat(tt, xds_obj, clr = "black", add = FALSE)
```

## Arguments

- tt:

  a set of time points

- xds_obj:

  a **`ramp.xds`** model object

- clr:

  the line color(s)

- add:

  if FALSE, use plot to draw new axes

## Value

an **`xds`** model object
